# 📌 Overview

DVLD is a desktop application designed to manage:

- Driver licenses
- International licenses
- Driving tests
- License applications
- Drivers & people records
- Users & permissions

Built using clean layered architecture and object-oriented design principles.

---

# 🏗️ System Architecture

```text
Presentation Layer
        ↓
Business Layer
        ↓
Data Access Layer
        ↓
Database
```
